# Product Overview

DNS IP+ is a Home Assistant custom component that retrieves public IP addresses via DNS queries. It supports both IPv4 and IPv6 address resolution using configurable DNS resolvers (default: OpenDNS).

The integration provides sensor entities that poll DNS servers to determine the current public IP address, useful for dynamic DNS monitoring and network diagnostics.

## Key Features

- IPv4 and IPv6 address resolution
- Configurable DNS resolvers and ports
- Config flow UI for easy setup
- Cloud polling integration type
- HACS compatible distribution
